# Real-Time-Linux-OS-Using-PREEMPT_RT-Kernel-with-5G-Log-Transfer
Real-Time Linux OS with 5G Log Transfer is a low-latency edge computing system developed on Raspberry Pi 4 using Buildroot to create a lightweight custom OS. Due to Buildroot limitations, including lack of package management and unsupported software for the SIM8200EA-M2 5G HAT module, the system was migrated to Raspberry Pi OS, where the PREEMPT_RT kernel was manually configured and cross-compiled to achieve deterministic real-time performance. The system integrates the SIM8200EA-M2 5G HAT for 5G connectivity and transmits live system logs to an MEC server for real-time web-based monitoring.
