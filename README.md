# Real-Time-Linux-OS-Using-PREEMPT_RT-Kernel-with-5G-Log-Transfer
This project develops a Real-Time Linux edge computing system using a PREEMPT_RT-enabled Raspberry Pi 4 and Private 5G Standalone (SA) network to achieve low-latency and deterministic communication. Live system logs are captured and transmitted over UDP to a Multi-access Edge Computing (MEC) server, where they are displayed in real time through a Flask and Socket.IO web interface. The system reduces scheduling latency and enables efficient monitoring for applications such as industrial automation, healthcare, and smart infrastructure.
